# 1. 机器学习基础

## 1.1 基本概念

1. 模型： 本质是“输入到输出的映射”，有两种表达形式：**确定性函数** 和 **条件概率分布**。一个是「硬答案」，一个是「软概率」。前者可以看作是后者概率为1/0的极端形式。
2. 模型的基本假设：数据是独立同分布产生的，确保具有规律性。
3. 模型的假设空间：模型所属的函数集合；例如：一维线性模型（**y = kx + b**）的**假设空间**，就是**所有不同 k（斜率）、b（截距）组合出来的直线的全体集合**。
4. 模型的选择准则：从模型的假设空间中挑选模型的打分准则。

## 1.2 分类

### 1.2.1 基本分类

1. 无监督学习：无标注数据
2. 有监督学习：有标注数据
3. 半监督学习：少量标注数据
4. 强化学习：通过奖励信号，在试错中不断优化策略

### 1.2.2 其他分类

1. 在线学习(一次接受一个样本)和批量学习(一次接受全部样本)
2. 贝叶斯学习(概率视角)和核方法(空间视角)

## 1.3 三要素

1. 模型：如1.1所述
2. 策略：选择最优模型的准则，包含以下概念：
    - **损失函数：**衡量 “预测错了多亏”(常见的0/1损失、平方损失和绝对值损失)
    - **经验风险**：训练数据的平均损失
    - **结构风险**：经验风险 + 模型复杂度惩罚，就是**正则化**：`R_srm(f) = 经验风险 + λJ(f)`
3. 算法：怎么求解模型

# 2. 过拟合与应对策略

## 2.1 概念

1. 过拟合：模型太复杂，对训练数据准、对新数据废；
2. 泛化能力：模型对未知数据的表现；泛化误差上界告诉我们：
    - 数据越多，泛化误差越小；
    - 模型越简单（假设空间越小），泛化误差越小。
3. 生成模型与判别模型：生成模型是先搞懂「数据是怎么来的」，再去做预测；判别模型是不管数据怎么来的，直接学「怎么区分不同类别」；

## 2.2 策略
1. 救星 1：正则化（结构风险最小化）
    - **L2 正则化（岭回归）**：给权重加`λ||w||²`，不让配料量太极端，配方变平滑。
    - **L1** 正则化（Lasso）：给权重加`λ||w||₁`，直接把没用的特征（比如 “杯子颜色”）权重压到 0，简化模型。(奥卡姆剃刀—— 如无必要，勿增实体)

2. 救星 2：交叉验证：数据不够时，反复用数据测模型：
	- 简单交叉验证**：70% 数据训练，30% 测试。**
	- **S 折交叉验证**：数据分 S 份，轮流用 S-1 份训练、1 份测试，取平均误差最优。
	- **留一交叉验证**：S折交叉验证的极端情形，当`S = N`时即为留一交叉验证。
