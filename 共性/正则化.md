# 1. L2正则化(最常用)

1. 核心思想：惩罚太大的权重，让所有参数都 “温和一点”；
2. 例子：过拟合时会将噪声点也一并学习进去，导致拟合曲线歪七扭八；L2正则化就像一根皮筋，强行把这根拟合曲线拉平拉正，避免其过分扭曲；

# 2. L1正则化(Lasso 回归)

1. 核心思想：直接把没用的特征权重压到 0，相当于**自动删掉无关特征**，让模型变简单。
2. 例子：想判断某种水果是不是西瓜，过拟合可能会将“摆放位置”这种无意义的特征也一并学习进去。L1 正则化会直接把 “摆放位置” 这类没用特征的权重归 0，避免被带偏；

# 3. Dropout(随机失活，神经网络专属)

1. 核心思想：训练时随机 “关掉” 一部分神经元，不让模型依赖固定几个神经元。
2. 例子：小组做项目，过拟合就是每次都靠那两个学霸干活，其他人躺平，换个项目这组直接崩。Dropout 会在每次干活随机抽走几个人，逼所有人都得会做事，不能依赖固定搭档。最后模型整体更稳，不会只记住局部套路。

# 4. 早停法

1. 核心思想：控制训练时间，在看到验证集效果开始变差就立刻停止。
2. 例子：刷题备考时，刷太少 → 学不会（欠拟合）；刷太多 → 把题目答案、排版都背下来了，只会这套题（过拟合）；早停 → 刷到刚好掌握方法，立刻停手，不钻牛角尖。

# 5. 数据增强

1. 核心思想：给模型看更多样的数据，让它见多识广，不靠死记。
2. 例子：教模型认 “猫”：只给它看 10 张白猫正面照 → 它会觉得 “只有白猫正面才是猫”（过拟合）；数据增强相当于给给模型看黑猫、橘猫、趴着的猫、侧脸猫、糊一点的猫 → 它学会猫的通用特征，不会被特例限制。
